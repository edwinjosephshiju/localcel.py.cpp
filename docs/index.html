🚀 Architecture Overview

Localcel.py.cpp replaces bloated frameworks with a lightning-fast, native Win32 C++ bootstrapper. The entire Python PyQt6 GUI application is securely embedded inside the `.exe` as an `RCDATA` resource.

  • No Heavy Frameworks: Built using C++20 and standard Win32 API without .NET or MFC.
  • Intelligent Dependencies: Natively checks for Python, `winget`, `cloudflared`, and `gh` CLI. It bypasses installations if it detects existing environments.
  • Native UI: Features a beautifully crafted, dark-mode GDI+ loading screen to mask dependency setups.
  • Supervisor Watchdog: Uses Windows Job Objects (`JOB_OBJECT_LIMIT_KILL_ON_JOB_CLOSE`) to absolutely guarantee no orphan processes remain when the app closes.

💻 Build Instructions

Compilation requires zero configuration files.

For Windows:

Run the native C++ build script from the x64 Native Tools Command Prompt for VS 2022:

mkdir dist
cl /EHsc build.cpp /Fe:dist\build.exe
dist\build.exe

For Linux:

mkdir -p dist
g++ -std=c++20 build.cpp -o dist/build
./dist/build

The output will be a standalone executable ready for deployment in the dist directory.
